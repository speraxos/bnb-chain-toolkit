Airdrops distribute free tokens to wallet addresses, often rewarding early users of protocols. This guide explains how to find, qualify for, and safely claim airdrops.

## What Are Crypto Airdrops?

Airdrops are token distributions where projects give away tokens for free. They serve several purposes:

- **Marketing**: Build awareness and community
- **Decentralization**: Distribute governance tokens widely
- **Rewarding Users**: Thank early adopters
- **Bootstrapping Liquidity**: Get tokens in circulation

### Types of Airdrops

| Type | Description | Example |
|------|-------------|---------|
| Retroactive | Rewards past users | Uniswap, Arbitrum |
| Holder | Requires holding a token | Cosmos ecosystem |
| Standard | Sign up or follow | New projects |
| Exclusive | For specific communities | NFT holders |

## Airdrop Farming Strategy

### 1. Use Promising Protocols Early

Target protocols that:
- Have significant funding (VC backing)
- Don't have a token yet
- Seem likely to decentralize via token

**Examples to watch:**
- New L2s and L3s
- Social protocols
- DEXs without tokens
- Wallet providers

### 2. Interact Meaningfully

Don't just make one transaction:
- Use multiple features
- Transact across different months
- Try different amounts
- Provide liquidity if possible
- Use testnet and mainnet

### 3. Maintain Multiple Wallets

Diversify across wallets:
- Different activity patterns
- Separate purposes
- But don't overdo it (Sybil detection)

**Warning**: Many protocols actively filter out Sybil attackers (users with many wallets doing identical actions).

### 4. Participate in Governance

If there's governance:
- Vote on proposals
- Create forum posts
- Join Discord discussions
- Contribute to documentation

### 5. Be a Power User

Stand out through:
- High volume
- Long history
- Cross-protocol usage
- Community involvement

### Red Flags (Scams)

**Never trust airdrops that:**
- Ask for seed phrase
- Require upfront payment
- Appear in your wallet unsolicited
- Come via DM from "support"
- Have too-good-to-be-true value

### Common Scam Patterns

**Fake Token Airdrop:**
1. Random token appears in wallet
2. Website promises high value
3. "Claim" requires wallet connection
4. Malicious contract drains wallet

**Defense**: Never interact with unknown tokens

**Phishing DM:**
1. "Support" DMs about airdrop
2. Links to fake claim site
3. Asks for wallet connection or seed

**Defense**: Real support never DMs first

## Claiming Airdrops Safely

### Before Claiming

1. **Verify official source** - Check project's official Twitter/Discord
2. **Check contract** - Verify on Etherscan it matches official
3. **Use new wallet first** - Claim with empty wallet if uncertain
4. **Revoke after** - Remove approvals at revoke.cash

### Claiming Process

1. Connect wallet to **official** claim site
2. Check eligibility
3. Review transaction details
4. Sign claim transaction
5. Receive tokens
6. Revoke any approvals granted

### Tax Implications

In most jurisdictions:
- Airdrops are taxable as income
- Value at time of receipt = taxable amount
- Keep records of claim date and value
- Subsequent sale = capital gains/loss

## Points Programs (2026 Meta)

Many protocols use points before token launch:

### How Points Work
1. Use protocol to earn points
2. Points track your activity
3. Points convert to tokens at launch
4. More points = more tokens

### Active Points Programs

| Protocol | Points | Activity Rewarded |
|----------|--------|-------------------|
| Various L2s | Ongoing | Bridging, DEX usage |
| Lending protocols | Ongoing | Deposits, borrows |
| Social apps | Ongoing | Engagement |

### Points Strategy
- Focus on high-TVL protocols
- Diversify across opportunities
- Don't overextend (gas costs add up)
- Consider opportunity cost

## Costs and Risks

### Financial Costs
- Gas fees for transactions
- Opportunity cost of locked funds
- Time investment

### Risks
- Protocol could fail
- No guarantee of airdrop
- Sybil detection = disqualification
- Tax liability even if token dumps

### Risk Management
- Only use funds you can afford to lose
- Diversify across protocols
- Don't over-leverage for farming
- Consider it a bonus, not income

## Anti-Sybil Measures

Projects increasingly fight airdrop farmers:

### Detection Methods
- Cluster analysis (wallets that interact together)
- Funding source tracing
- Identical timing/amounts
- GitHub/Discord linking
- Passport systems (Gitcoin Passport)

### How to Stay Legitimate
- Use natural activity patterns
- Don't fund wallets from same source
- Vary transaction amounts
- Space out activity
- Actually use protocols genuinely
