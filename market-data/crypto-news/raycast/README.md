## Installation

### From Raycast Store (Coming Soon)

Search for "Free Crypto News" in Raycast.

### Manual Installation

1. Clone the repository
2. Navigate to the raycast directory
3. Run `npm install && npm run dev`

## Actions

Each news item supports:

- **Open in Browser** - Read the full article
- **Copy Link** - Copy the article URL
- **Copy Title** - Copy the headline

## Privacy

This extension:
- Only fetches from `cryptocurrency.cv`
- Does not track you
- Does not collect any data
- Open source
