# Development Scripts

Utility scripts for development and release.

## Available Scripts

| Script | Description |
|--------|-------------|
| `build.sh` | Build the package |
| `release.sh` | Release to PyPI |
| `docs.sh` | Build and serve documentation |
| `lint.sh` | Run linters and formatters |
| `test.sh` | Run test suite |
| `fetch_abis.py` | Fetch real-world ABIs for testing |

## Usage

All scripts should be run from the repository root:

```bash
./scripts/test.sh
./scripts/lint.sh
./scripts/build.sh
```
