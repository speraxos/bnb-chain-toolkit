## Tool Organization

All tools are organized by:

1. **Package Category** - DeFi, Wallets, Trading, etc.
2. **Protocol** - Specific blockchain or protocol (Aave, Uniswap, etc.)
3. **Function** - Read operations vs Write operations
4. **Complexity** - Simple queries vs Multi-step workflows

## Using Tools

### Basic Usage

```typescript
import { McpClient } from '@universal-crypto/mcp-client';

const client = new McpClient({
  serverUrl: 'http://localhost:3000',
});

// List all available tools
const tools = await client.listTools();

// Call a specific tool
const result = await client.callTool('get_token_price', {
  token: 'ethereum',
  currency: 'usd',
});
```

### Tool Discovery

```typescript
// Search for tools by keyword
const tradingTools = tools.filter(t => 
  t.name.includes('swap') || t.description.includes('trade')
);

// Filter by category
const defiTools = tools.filter(t => 
  t.category === 'defi'
);
```

### Paid Tools (x402)

Some tools require payment via the x402 protocol:

```typescript
// Tools automatically handle payment
const result = await client.callTool('deep_token_analysis', {
  token: '0x...',
});

// Payment proof is included in headers
// Client auto-pays if wallet configured
```
